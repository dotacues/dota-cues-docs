# Dota Cues: Public Documentation

This repository hosts the public-facing documents for the
[Dota Cues](https://github.com/dotacues/dota-cues) Overwolf app.
Content is published via GitHub Pages.

## Pages

- [Privacy Policy](https://dotacues.github.io/dota-cues-docs/privacy-policy)
- [Terms of Service](https://dotacues.github.io/dota-cues-docs/terms-of-service)

## Why a separate repo

The main app source lives in a private repository. These documents need
public URLs for the Overwolf store submission, so they live here as a
separate, public companion repo.

## Editing

Edit the markdown files in this repo's root and push to `main`. GitHub
Pages rebuilds automatically (~30 seconds). The published URLs above
update on the next render.
